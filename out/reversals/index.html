Who changed their mind?

Across HoldLens’s 8-quarter archive, 55 reversals are comebacks — the manager exited the name and then opened a fresh position later. 173 are whipsaws — the thesis broke within a few quarters of the entry.

Dataroma doesn’t mark either. HoldLens keeps the action history, so a manager walking BACK an exit is visible as what it is: the second-hardest decision in professional investing, after the first entry. Comebacks are rare and loud.

How reversals are detected

new/exit action sequence, same manager × ticker

  • 1. Action filter. Only new and exit actions. Adds and trims don’t count — a true reversal is going to zero and back.
  • 2. Comeback. The first event is an exit and a later event is a new. The manager sold it completely, then later opened a fresh position in the same name. Rarest pattern in the archive.
  • 3. Whipsaw. The first event is a new and a later event is an exit. The manager opened the position and then walked within the 8-quarter window.
  • 4. Gap quarters. Distance between the first and last event in the sequence. Larger gap = more distance between the two decisions.

The archive only covers 8 quarters — reversals that span longer windows are not visible yet. As more quarters land, the comeback count will grow.

Not investment advice. Reversal detection is limited by archive coverage. Methodology.
